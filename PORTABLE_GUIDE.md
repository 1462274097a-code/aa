# 🚀 便携式 exe 使用指南

## 方案一：使用现有安装程序（推荐）

虽然当前版本是 NSIS 安装程序，但你可以：

### 方法 1：安装后提取便携版本
1. 运行安装程序：`Storyboard-Copilot_0.1.7_x64-setup.exe`
2. 安装到任意目录（比如 `C:\Program Files\分镜助手`）
3. 直接复制整个安装目录到 U盘或其他位置
4. 运行目录中的 `分镜助手.exe` 即可使用

### 方法 2：使用安装程序的便携模式
安装程序通常提供：
- **安装模式**：正常安装到系统
- **便携模式**：解压到指定目录，无需安装

---

## 方案二：从源码构建真正的便携式 exe

如果你需要真正的便携式版本，可以：

### 在本地构建便携式 exe

```bash
# 1. 进入项目目录
cd F:/agents/Storyboard-Copilot-0.1.6

# 2. 安装依赖（如果还没安装）
npm install

# 3. 构建前端
npm run build

# 4. 构建 Tauri 应用（生成原始 exe）
cd src-tauri
cargo build --release

# 5. 便携式 exe 就在这里！
# F:/agents/Storyboard-Copilot-0.1.6/src-tauri/target/release/分镜助手.exe
```

**这个 exe 可以直接运行，无需安装！**

### 便携式包内容
将以下文件复制到一起：
- `分镜助手.exe` （主程序）
- 所有 `.dll` 文件（依赖库）
